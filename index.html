    Guías de Ejercicios

    Guía 1

    Introducción a HTML y CSS

    Esta guía introduce los conceptos fundamentales de HTML5 y CSS3, enseñando la estructura básica de documentos web y las diferentes formas de aplicar estilos.

    CONTENIDOS DE LA GUÍA:

    • Estructura HTML y sintaxis CSS
    • Métodos para incluir CSS (embebido, interno, externo)
    • Selectores básicos (universal, etiqueta, clase, ID)
    • Propiedades de texto y fuentes

    Guía 2

    Selectores CSS Avanzados y Modelo de Caja

    Esta guía profundiza en técnicas avanzadas de selección de elementos HTML y explora el modelo de caja de CSS para crear diseños más complejos.

    Contenidos:

    • Selectores avanzados y combinación de selectores
    • Pseudo-elementos y pseudo-clases
    • Modelo de caja (margin, border, padding, content)
    • Elementos de bloque vs elementos de línea

    Guía 3

    Posicionamiento, Visualización y Maquetación

    Esta guía enseña las técnicas de posicionamiento CSS y control de visualización para crear layouts complejos y menús de navegación.

    Contenidos:

    • Tipos de posicionamiento CSS
    • Propiedades de visualización (display, visibility, overflow, z-index)
    • Propiedad float y clear
    • Diseños fijos y líquidos/fluidos
    • Creación de menús multinivel

    Enlaces y ejemplos:

    • Ejemplo Agencia de viajes
    • Ejemplo Z-index
    • Ejemplo Posicionamiento
    • Ejemplo Menú horizontal

    Guía 4

    Introducción al Responsive Design

    Esta guía introduce el diseño web responsivo para crear sitios que se adaptan a diferentes dispositivos y tamaños de pantalla.

    Contenidos:

    • Concepto de Responsive Web Design
    • Meta tag viewport y diseño fluido
    • Media queries y breakpoints
    • Estrategias Mobile First vs Desktop First
    • Layouts responsivos completos

    Enlaces y ejemplos:

    • Ejemplo Agencia de viajes
    • Ejemplo Z-index
